# Android-MVP-Tutorial
Tutorial for MVP in Android with Java. An app for simple generic Login

# Technologies used
1. [Retrofit](http://square.github.io/retrofit/)
2. [RxJava2](https://github.com/ReactiveX/RxJava)
3. [Dagger2](https://github.com/google/dagger)
4. [Gson](https://github.com/google/gson)
5. [ButterKnife](jakewharton.github.io/butterknife/)
